# 🌾 Kachin Farming Web (ကချင်ပြည်နယ် စိုက်ပျိုးရေးနည်းပညာ)

ဒီ Website ဟာ ကချင်ပြည်နယ်ရှိ တောင်သူဦးကြီးများအတွက် **၁.၂၅ ဧက ပဲငပိ (ပဲပုပ်)** စိုက်ပျိုးရေးနှင့် နည်းပညာပေး အချက်အလက်များကို ဝေမျှပေးရန် ရည်ရွယ်ပါသည်။

## 🚀 Features (လုပ်ဆောင်ချက်များ)
* **Crop Calendar:** ဇန်နဝါရီမှ မေလအထိ ပဲပုပ်စိုက်ပျိုးမှု အဆင့်ဆင့်။
* **Geography Focus:** ကချင်ပြည်နယ်၏ မြေဆီလွှာနှင့် ရာသီဥတုဆိုင်ရာ အကြံပြုချက်များ။
* **Farming Calculator:** ဧကအလိုက် မြေဩဇာနှင့် ဆေးနှုန်းထား တွက်ချက်စက်။
* **Organic Lab:** သဘာဝပိုးသတ်ဆေးနှင့် NPK စပ်နည်းလမ်းညွှန်များ။

## 📍 Tech Stack (အသုံးပြုထားသော နည်းပညာ)
* **Frontend:** HTML5, CSS3, JavaScript
* **Hosting:** Vercel
* **Version Control:** GitHub

## 📅 Project Timeline
* **Start Date:** January 2, 2026
* **Location:** Kachin State (ကချင်ပြည်နယ်)
* **Goal:** ၁.၂၅ ဧကမှ အထွက်နှုန်းအကောင်းဆုံး ရရှိရန်။
